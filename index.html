CTF Writeups

🚩 Phishing Log Analysis Challenge

Problem Statement

Identify phishing indicators in SIEM logs from a simulated enterprise environment. The challenge involves analyzing CEF-formatted logs to detect a targeted phishing campaign.

Reconnaissance

  • Analyzed CEF formatted email logs spanning 48-hour period
  • Identified suspicious sender domains with typosquatting patterns
  • Discovered unusual email headers with encoded payloads
  • Traced email flow through multiple SMTP relays

Exploitation & Detection

Created custom Wazuh decoder to parse email headers and extract key indicators:

CEF src=(\S+).*suser=(\S+).*msg=(.*?)$ srcip,sender,subject

Built correlation rule for automated detection:

100499 suspicious_domain|encoded_payload Phishing attempt detected

Tools Used

Wazuh Regex Linux CEF Parsing Python

Key Lessons Learned

  • Log normalization is critical - Converting various log formats to a unified schema enables better correlation
  • Alert tuning reduces false positives - Proper threshold configuration decreased alerts by 75%
  • Context matters - Email metadata analysis provided crucial indicators beyond just content inspection
  • Automation scales detection - Custom decoders enabled real-time threat identification

🚩 Web Server Compromise Analysis

Problem Statement

Investigate suspicious activity on a web server and determine the attack vector, persistence mechanism, and data exfiltration methods used by the attacker.

Investigation Steps

  • Analyzed Apache access logs for SQL injection attempts
  • Identified webshell upload via file upload vulnerability
  • Traced lateral movement through auth logs
  • Discovered data staging directory with compressed archives

Key Findings

Attack timeline reconstruction using Splunk queries:

index=web sourcetype=apache_access | rex field=_raw "(?UNION.*SELECT)" | stats count by src_ip, exploit | where count > 5

Lessons Learned

  • Timeline analysis reveals attack progression
  • Multiple data sources provide complete picture
  • Persistence mechanisms require thorough investigation
